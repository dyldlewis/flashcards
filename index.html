Javascript

?

The programming language of the web. The only programming language that all web browsers recognize.

Operators

?

Operators are the basic symbols used to perform operations, most often arithmetic.

Variables

?

Containers for storing data values.

Variable Naming Conventions

?

Containers for storing data values.

Functions

?

A JavaScript function is a block of code designed to perform a particular task.

Methods

?

JavaScript methods are the actions that can be performed on objects. A JavaScript method is a property containing a function definition.

Arguments

?

An argument is the data you pass to a function.

Parameters

?

The default values passed to a function, replaced by arguments.

Return

?

The return statement stops the execution of a function and returns a value from that function.

Chaining Methods

?

Chaining Methods, also known as Cascading, refers to repeatedly calling one method after another on an object, in one continuous line of code.

Strings

?

Any combination of keyboard characters contained within quotations.

Booleans

?

The Boolean object represents two values, either "true" or "false".

Undefined

?

Undefined is used to represent the absence of some value.

NaN

?

Not a number.

Escape

?

This function makes a string portable, so it can be transmitted across any network to any computer that supports ASCII characters.

Alert

?

The alert() method displays an alert box with a specified message and an OK button.

Comments

?

JavaScript comments can be used to explain JavaScript code, and to make it more readable, as well as prevent execution while testing code.

Jquery

?

A javascript library used to easily add animation and effects to webpages.

Attributes

?

Attributes provide additional information about an element.
